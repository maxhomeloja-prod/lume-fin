Política de Segurança da Informação e Privacidade de Dados
Lume Gestão Empresarial — Versão 1.0, Junho de 2026

1. Introdução e Escopo

Esta Política de Segurança da Informação e Privacidade de Dados ("Política") estabelece os princípios, diretrizes e controles adotados pela Lume Gestão Empresarial ("Lume") na proteção dos dados pessoais e empresariais processados através do sistema Lume Financeiro, incluindo suas integrações com plataformas de marketplace (Shopee, TikTok Shop, Mercado Livre e similares).

O Lume é uma plataforma de gestão empresarial multi-tenant que oferece módulos financeiros, de produção e de integração com canais de venda para empresas do setor de manufatura e comércio. Esta Política se aplica a todos os dados processados em nome dos clientes ("Vendedores") que utilizam o sistema.

2. Segurança da Informação

2.1 Arquitetura e Infraestrutura
O Lume opera sobre infraestrutura de nuvem gerenciada (Cloudflare Workers e Supabase/AWS), com proteção nativa contra ataques de negação de serviço (DDoS), criptografia em repouso, e toda comunicação criptografada em trânsito via TLS/HTTPS.

2.2 Controle de Acesso
O acesso aos dados é regido pelo princípio do menor privilégio: autenticação via Supabase Auth com senhas criptografadas, Row-Level Security (RLS) no banco de dados garantindo isolamento entre empresas (tenants), autenticação adicional por PIN para acesso via WhatsApp com bloqueio automático após tentativas inválidas, e segredos de integração armazenados de forma criptografada, nunca em código-fonte.

2.3 Classificação e Proteção de Dados
Os dados são classificados em operacionais, financeiros e pessoais de terceiros, cada categoria com controles proporcionais à sua sensibilidade, tratados conforme a Lei Geral de Proteção de Dados (LGPD).

2.4 Gerenciamento de Vulnerabilidades
O patching de infraestrutura é gerenciado automaticamente pelos provedores (Cloudflare, Supabase). Dependências de software são mantidas atualizadas e logs de aplicação são monitorados para identificação de comportamento anômalo.

3. Privacidade e Proteção de Dados Pessoais

3.1 Finalidade do Tratamento
Os dados pessoais são utilizados exclusivamente para viabilizar a gestão comercial, financeira e de produção dos Vendedores, incluindo a sincronização de pedidos de marketplaces integrados. Não há comercialização ou uso para finalidades publicitárias.

3.2 Armazenamento e Localização dos Dados
Os dados são armazenados em infraestrutura gerenciada (Supabase), hospedada em data centers nos Estados Unidos e/ou Brasil.

3.3 Direitos do Titular de Dados
Em conformidade com a LGPD, garantimos aos titulares: acesso, correção, exclusão e portabilidade de dados pessoais. O Lume auxilia Vendedores e plataformas integradas (incluindo TikTok Shop) no atendimento a essas solicitações dentro de prazo razoável.

3.4 Retenção e Exclusão de Dados
Os dados são retidos durante a vigência contratual. Ao término da relação, mediante solicitação formal, todos os dados do Vendedor serão excluídos, salvo obrigação legal de retenção.

4. Gestão e Resposta a Incidentes de Segurança

4.1 Processo de Resposta a Incidentes
1. Identificação — monitoramento contínuo de logs.
2. Contenção — isolamento imediato do vetor de comprometimento.
3. Avaliação de impacto — determinação do escopo afetado.
4. Comunicação — notificação a Vendedores afetados e, quando exigido, à ANPD e plataformas integradas, dentro dos prazos legais.
5. Remediação — correções técnicas para prevenir recorrência.

4.2 Histórico de Incidentes
Até a data de publicação desta Política, o Lume não registrou violações de segurança que tenham resultado em exposição de dados pessoais, nem recebeu reclamações de autoridades reguladoras relacionadas ao tratamento de dados.

5. Contato, Governança e Atualizações

5.1 Responsável pela Proteção de Dados (DPO)

5.2 Revisão e Atualização
Esta Política é revisada no mínimo anualmente, ou sempre que houver alteração relevante na arquitetura técnica ou na legislação aplicável.

5.3 Legislação Aplicável
Esta Política é regida pela Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018 — LGPD) e demais normas aplicáveis no território brasileiro.
